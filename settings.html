Settings for Browser Session History Display Extension

This setting determines the length of inactivity (in minutes) required before the extension determines the start of a new browsing session. For example, if you set this to 5, any gap of 5 minutes or more between site visits will create a new session.

This tells the extension how far back (in hours) to go when collecting and displaying your browser history. If you set this to 24, the extension will only show data from the past 24 hours. If this value is set too high then the extension startup and updates may become unacceptably slow.

When checked, the extension will retrieve the small icon (favicon) associated with each site via Google's favicon service. Uncheck if you prefer not to use this feature or if you encounter loading/performance issues.
